Welcome to Momoda API 2.0's documentation
==========================================

|

Momoda API is a set of javascript-like, executed-at-runtime scripts runs inside Momoda system. Momoda API provides operations on object, UI, behavior and effect in 3D scene.  Momoda API could be used to make customized, industries specific apps such as storage management, emergency drilling, archive management, intelligent buildings etc.

Momoda API supports fetch and interact with external data source. Through Momoda API's data interface, 3rd party system could push realtime data to 3D scene to drive realtime object behavior and/or information display.
